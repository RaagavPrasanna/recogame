# RecoGame service

## Setup

1. Create a `.env file` in root directory
    ```bash
    MONGO_CONNECTION_URI="mongodb+srv://<USERNAME>:<PASSWORD>..."
    ```
2. Install all the dependencies
    ```bash
    cd server && npm run install && cd ..
    cd client && npm run install && cd ..
    ```
3. Run the server
    ```bash
    npm run start
    ```
